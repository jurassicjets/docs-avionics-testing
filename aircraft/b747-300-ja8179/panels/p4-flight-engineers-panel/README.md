# P4 — Flight Engineer's Panel

> _The flight engineer's station on JA8179: fuel, electrical, hydraulic, pneumatic, and APU systems management, plus one of the three CIVA INS CDUs._

Panel within [`b747-300-ja8179`](../../README.md).

## Location

Flight engineer's (second officer's) station, aft of the first officer on the right side of the flight deck. For an annotated cockpit overview see the parent aircraft's [`cockpit-map.md`](../../cockpit-map.md).

Designated **P4** per the 747 WDM (the fuel pages, 28-41-11, label it "P4 — Flight Engineer's Panel"). An earlier repo draft indexed this panel as P9; P9 is actually a pedestal panel.

## Modules

Functional groupings within this panel. Each links to a module folder.

| Module | Description | Status |
|---|---|---|
| [m154-fuel-control](m154-fuel-control/README.md) | fuel quantity indicators, totalizer, gross-weight-select, gauge test (ATA 28-41) | in progress |
